⚡ Naruto Jutsu Hand Tracking ⚡

A gesture-based project inspired by Naruto jutsu techniques. It uses hand tracking (MediaPipe) and a webcam to trigger anime-style effects like Rasengan and Chidori.

🌪️ Rasengan (Right Hand)

Description:
When the right hand opens, the system detects the gesture and displays a Rasengan energy effect near the hand.

Key Features

Real-time hand tracking using MediaPipe

Detects right-hand open gesture

Displays animated Rasengan effect

Effect strength changes with hand movement

Benefits / Use Cases

Demonstrates computer vision and gesture control

Interactive AR-style webcam project

Useful for learning AI hand tracking

How to Use

Start the webcam.

Show your right hand to the camera.

Open your hand to activate the Rasengan effect.

⚫ Chidori (Left Hand)

Description:
When the left hand opens, the system triggers a Chidori lightning effect near the detected hand position.

Key Features

Detects left-hand gestures

Lightning animation effect

Real-time overlay tracking

Smooth fade in/out animation

Benefits / Use Cases

Demonstrates gesture-based interaction

AR visual effect experimentation

Fun Naruto-themed AI project

How to Use

Show your left hand to the webcam.

Open your hand.

The Chidori lightning effect appears near the hand.
